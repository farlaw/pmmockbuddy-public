# Architecture

PM Mock Buddy is an AI-assisted PM interview practice tool.

## High-level flow

1. User selects an interview category
2. App generates or selects a realistic PM interview prompt
3. User drafts or speaks an answer
4. AI evaluates the response across structure, clarity, assumptions, tradeoffs, and depth
5. User iterates based on feedback
6. Practice history helps identify recurring improvement areas

## Core modules

- Prompt generation
- Answer evaluation
- Feedback and coaching
- Behavioral story refinement
- Practice history
- Iteration loop

## Notes

This public repo is a sanitized portfolio version. It does not include API keys, tokens, private transcripts, user data, or personal job-search notes.
